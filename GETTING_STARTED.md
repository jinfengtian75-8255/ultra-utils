# 🚀 시작 가이드 (Getting Started)

안녕하세요! 이 문서는 **VS Code(Visual Studio Code)** 를 처음 사용하시거나, 프로젝트 실행에 어려움을 겪으시는 분들을 위한 가이드입니다.

## 1. VS Code 화면 설명

지금 보고 계신 이 프로그램이 바로 **VS Code**입니다. 크게 세 부분으로 나뉩니다:

*   **왼쪽 사이드바 (Activity Bar)**: 파일 탐색기(문서 모양 아이콘), 검색, 확장 프로그램 등을 열 수 있는 메뉴입니다.
*   **가운데 편집창 (Editor)**: 코드를 작성하고 수정하는 메인 화면입니다.
*   **하단 패널 (Panel)**: 터미널(명령어 입력창), 문제(에러 메시지) 등을 확인하는 곳입니다.

## 2. 터미널(Terminal) 열기

명령어를 입력하려면 **터미널**을 열어야 합니다. 다음 중 편한 방법을 사용하세요.

1.  **단축키**: `Ctrl` + `Swung Dash(~)` (물결표/백틱 키, 보통 Tab 키 위에 있습니다)
2.  **상단 메뉴**: `Terminal` (터미널) > `New Terminal` (새 터미널)

터미널이 열리면 하단에 새로운 창이 나타나고, 깜빡이는 커서가 보일 것입니다.

## 3. 프로젝트 설치 및 실행

터미널에 다음 명령어들을 **한 줄씩 입력하고 엔터(Enter)** 를 누르세요.

### 1단계: 설치 (처음 한 번만)
필요한 프로그램(패키지)들을 설치합니다.
```bash
npm install
```
> **주의**: 만약 빨간색 에러 메시지가 나온다면, 권한 문제일 수 있습니다. 그럴 땐 아래 명령어를 먼저 입력해 보세요:
> `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`

### 2단계: 실행
설치가 끝나면 서버를 켭니다.
```bash
npm run dev
```

### 3단계: 확인
"Ready" 또는 "started server on ..." 메시지가 나오면 성공입니다!
이제 인터넷 브라우저(크롬 등)를 켜고 아래 주소로 접속하세요:
👉 **[http://localhost:3000](http://localhost:3000)**

---

**팁**: 서버를 끄고 싶을 때는 터미널에서 `Ctrl` + `C`를 누르면 됩니다.
