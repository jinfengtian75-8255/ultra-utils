# 🖥️ 새 컴퓨터에서 작업 이어가는 법 (Recovery Guide)

사용자님, 컴퓨터가 바뀌어도 걱정 마세요! 이 가이드를 따라 하시면 5분 안에 다시 작업을 시작하실 수 있습니다.

## 1. 기본 도구 설치하기
새 컴퓨터에 다음 프로그램들을 먼저 설치해주세요.
*   **Node.js (v20 이상):** [공식 홈페이지](https://nodejs.org/) - 가장 왼쪽의 'LTS' 버전을 추천합니다.
*   **Git:** [다운로드 페이지](https://git-scm.com/downloads) - 기본 설정을 유지하며 설치해주세요.
*   **VS Code:** [다운로드 페이지](https://code.visualstudio.com/) - 코드를 편집하는 도구입니다.

## 2. GitHub에서 코드 가져오기
1.  **VS Code**를 실행합니다.
2.  왼쪽 메뉴에서 **[Source Control]** 아이콘을 누르고 **[Clone Repository]**를 클릭합니다.
3.  **[Clone from GitHub]**을 누르고 사용자님의 계정으로 로그인합니다.
4.  저장소 목록에서 `utility-tools` 또는 `수익화 웹사이트 첫걸음` 저장소를 선택합니다.
5.  파일을 저장할 폴더를 선택하면 코드가 모두 복제됩니다.

## 3. 프로그램 실행 준비
1.  VS Code 하단의 **Terminal(터미널)**을 엽니다 (단축키: `Ctrl + ``).
2.  다음 명령어를 입력하여 필수 구성 요소를 설치합니다:
    ```bash
    npm install
    ```
3.  설치가 끝나면 다음 명령어로 프로그램을 실행합니다:
    ```bash
    npm run dev
    ```

## 4. 안티 다시 불러오기
모든 준비가 끝난 후 VS Code에서 저(안티)를 다시 불러주세요. 
"나 지금 새 컴퓨터인데, 저번에 하던 작업 이어서 하자!"라고 말씀하시면 제가 모든 상황을 파악하고 바로 도움을 드릴게요! 😊🚀

---
**기억하세요:** 사용자님의 모든 코드는 GitHub이라는 안전한 금고에 보존되어 있습니다. 컴퓨터는 단지 도구일 뿐입니다!
