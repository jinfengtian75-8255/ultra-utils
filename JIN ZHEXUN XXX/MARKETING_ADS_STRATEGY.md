# 🎯 UltraUtils 구글 광고(Google Ads) 실전 전략 가이드

사장님, 이 파일은 구글 광고 관리창을 보면서 그대로 복사해서 사용하시라고 만든 '컨닝 페이퍼'입니다. 

---

## 💡 필수 세팅 원칙 (돈 아끼는 법)
1. **전문가 모드 사용**: 처음 가입 시 하단의 '전문가 모드로 전환'을 꼭 클릭하세요.
2. **캠페인 목표**: '웹사이트 트래픽' -> 캠페인 유형 '검색(Search)'
3. **키워드 형식**: 정확한 타겟팅을 위해 대괄호`[키워드]` 형식을 권장합니다.
4. **일일 예산**: 5,000원 ~ 10,000원으로 시작 (효과 확인 후 증액)

---

## 📂 도구별 광고 세팅 정보

### 1. 종합 (메인 페이지)
*   **최종 URL**: `https://ultra-utils.com`
*   **핵심 키워드**: `[무료 온라인 도구]`, `[크리에이터 유틸리티]`, `[무료 웹툴]`
*   **광고 제목 1**: 로그인 없는 100% 무료 도구 모음
*   **광고 제목 2**: 유튜버 필수! UltraUtils 슈퍼팩
*   **설명**: 가입 없이 3초 만에 끝내는 배경 제거, 이미지 보정, PDF 합치기. 작업 시간을 10배 단축하세요.

### 2. AI 배경 제거 (누끼 따기)
*   **최종 URL**: `https://ultra-utils.com/tools/background-remover`
*   **핵심 키워드**: `[누끼 따기 사이트]`, `[이미지 배경 제거]`, `[사진 배경 지우기]`
*   **광고 제목 1**: AI 누끼 따기 - 1초 만에 배경 제거
*   **광고 제목 2**: 설치 없는 고해상도 배경 지우기
*   **설명**: 고해상도 이미지 지원! AI가 머리카락 한 올까지 깔끔하게 지워드립니다. 서버 업로드가 없어 안전합니다.

### 3. 유튜브 플래너 (기획 도구)
*   **최종 URL**: `https://ultra-utils.com/tools/youtube-planner`
*   **핵심 키워드**: `[유튜브 주제 추천]`, `[쇼츠 아이디어]`, `[영상 시나리오 작성]`
*   **광고 제목 1**: 유튜브 떡상 기획, AI가 짜드립니다
*   **광고 제목 2**: 쇼츠 아이디어부터 제목 추천까지
*   **설명**: 조회수 터지는 제목과 태그 추천. 초보 유튜버도 전문가처럼 기획하세요. 기획서 .TXT 저장 가능.

### 4. 유튜브 썸네일 다운로더
*   **최종 URL**: `https://ultra-utils.com/tools/youtube-thumbnail`
*   **핵심 키워드**: `[유튜브 썸네일 다운로드]`, `[쇼츠 썸네일 추출]`, `[썸네일 고화질]`
*   **광고 제목 1**: 고화질 유튜브 썸네일 1초 추출
*   **광고 제목 2**: 쇼츠 수직 이미지 완벽 지원
*   **설명**: 링크 하나로 4K 고화질 썸네일 소장. 바로 AI 편집기로 연결되어 썸네일 수정이 가능합니다.

### 5. PDF 마스터
*   **최종 URL**: `https://ultra-utils.com/tools/pdf-master`
*   **핵심 키워드**: `[무료 PDF 합치기]`, `[PDF 페이지 순서 변경]`, `[PDF 나누기]`
*   **광고 제목 1**: PDF 합치기 & 나누기 - 로그인 없이
*   **광고 제목 2**: 페이지 순서 변경이 자유로운 PDF 도구
*   **설명**: 용량 제한 없는 무료 PDF 편집. 서버 업로드 없는 100% 로컬 처리로 소중한 기업 정보를 보호하세요.
